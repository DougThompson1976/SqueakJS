    To see SqueakJS in action visit Lively Squeak. Alternatively (especially if you are on a mobile device) try Simple Squeak.

    I'm currently working on loading more demanding images. The Lively Squeak interface supports dropping a Squeak image from your local disk into the world. E.g., the Etoys image loads, but does not run successfully yet. The Simple Squeak page works fine on an iPad, but not as well on Android.
