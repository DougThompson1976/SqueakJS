About

Squeak is a modern implementation of Smalltalk, the original dynamic object-oriented programming environment. It runs bit-identically on virtually any platform, and now in the web browser, too. Currently SqueakJS only works with some versions of Squeak, but it is intended to run the latest one in the future.

To see SqueakJS in full action visit Lively Squeak. Alternatively (especially if you are on a mobile device) try Simple Squeak. For a more demanding work load try Etoys. Finally, you can also run the automated Benchmarks on your machine.

Contributions are very welcome! For discussion please use the vm-dev Mailing List.

-- Bert Freudenberg, July 2014
